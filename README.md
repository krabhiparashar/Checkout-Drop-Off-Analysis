## Problem Statement
Users drop off during checkout. Goal is to identify where and why.

## Dataset
- ecommerce_events_large.csv
- Contains user actions (view, cart, purchase)

## Approach
- Data cleaning
- Funnel creation (Cart → Checkout → Payment → Purchase)
- Drop-off rate calculation

## Key Insights
- 65% drop at payment stage
- High cart activity but low conversion

## Recommendations
- Add COD option
- Improve payment UX
- Reduce page load time
